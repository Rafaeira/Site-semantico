As 10 perguntas mais realizadas sobre html e css em 2025

1. O que é Web Design e qual sua relação com Front-End?

Web design é o planejamento visual de interfaces digitais com foco em estética,usabilidade e experiência do usuário, a sua relação com o front end é muito
próxima já que em um projeto o web design projeta a interface visual do site e o dev front end executa esse planos.

2. Cite três princípios de design que melhoram a experiência do usuário.

1) Hierarquia visual, para que o usuário consiga compreender corretamente os elementos do site por exemplo oque é um título, um capítulo um rodapé etc.
2) Consistência no design, não podemos colocar uma cor de fundo na página inicial e uma outra completamente diferente na seção de login por exemplo, temos
que manter uma certa consistência para que o site fique visualmente agradável.
3) Espaçamento adequado, para que o usuário consiga indentificar onde começa e acaba um artigo de um site por exemplo, um espaçamento bem pensando dos elementos
pode facilitar essa compreesão fugindo de qualquer tipo de dúvida.

3. Qual a diferença entre Wireframe e Protótipo?

A principal diferença entre wireframe e protótipo é que enquanto o wireframe é utilizado para fazer um rascunho rápido de onde ficarão localizados cada
elemento do site o protótipo é feito para ser uma versão mais aprimorada e fiel ao resultado final da página sendo estilizado com cores, fontes, indentidade visual própria etc.

4. Liste três ferramentas usadas para criar protótipos.

Figma, Canva e Adobe XD.

5. Explique a função da tag header dê um exemplo de uso.

O elemento header é utilizado para o conteúdo introdutório de um documento ou artigo como cabeçalhos e menus de navegação do site.
Quando você vai colocar o título de uma página por exemplo é interessante que você utilize o header para demarcar que aquela é a parte introdutória da sua página.

6. Qual a diferença entre uma tag em bloco e uma tag inline?

A tag em bloco ocupa toda a largura disponível da linha, sempre começa em uma nova linha e pode conter outros elementos dentro dela. Já a tag inline
ocupa apenas o espaço necessário ao seu conteúdo, não quebra a linha automaticamente e geralmente não pode conter elementos de bloco dentro.

7. Como aplicar CSS externo em um documento HTML?

Para aplicar css externo você prescisa criar um documento .css e depois linkar ele na sua página html com o comando link type="text/css" rel="stylesheet" href="./endereço do arquivo"

8. Cite três propriedades CSS comuns e sua função.

Background color, é utilizado para definir a cor de fundo de algum elemento. Font-size é utilizado para definir o tamanho da letra na tela.
Margin é utilziado para definir o espaçamento externo ao redor de um elemento (distância entre elementos).

9. O que é DOM e como o JavaScript interage com ele?

DOM (Document Object Model) é uma representação do HTML como uma árvore de objetos.
Cada tag vira um nó/objeto que o JavaScript pode acessar e modificar. O JavaScript interage com o DOM selecionando os seus elementos e alterando textos, estilos(css), classes etc.

10. Escreva um exemplo simples de código JavaScript que exibe uma mensagem no console.

Para exibir uma mensagem no console usamos o comando: console.log("Mensagem que você deseja")
